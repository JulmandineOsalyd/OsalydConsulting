[Copilot de A à Z] - Peut-on créer des agents sans licence Copilot ?

Julie BREDECHE 15 janvier 2026

“Est-ce qu’on peut créer des agents Copilot sans licence ?”

La réponse courte : oui, on peut créer des agents déclaratifs sans licence Copilot payante, mais les possibilités restent limitées.

Ici on parle uniquement des licences Copilot pour Microsoft 365 versus Copilot Chat inclus dans les licences Microsoft 365 standards.

👉 Microsoft détaille d’ailleurs clairement les différences de fonctionnalités entre Copilot Chat (gratuit) et Copilot pour Microsoft 365 dans sa documentation officielle.

Cas n°1 : Pas de licences Copilot 

Le créateur de l'agent et les utilisateurs disposent de la version gratuite de Copilot Chat.

✅ Dans ce cas, tout utilisateur sans licence peut créer un agent déclaratif basé uniquement sur :

  • des sites web publics
  • de la documentation accessible en ligne

❌ Limite majeure : Aucun accès aux données de l'entreprise (tenant Microsoft, SharePoint, OneDrive...)

 

💡 Cas d'usages types : Veille, FAQ basée sur de la documentation publique

Cas n°2 : Licence Copilot pour le créateur

Le créateur de l'agent dispose d'une licence Copilot pour Microsoft 365, tandis que les utilisateurs disposent la version gratuite Copilot Chat.

✅ Ici, on passe un cap important. Le créateur peut créer un agent déclaratif basé sur :

  • des sites web publics
  • des fichiers fournis à l'agent (Word, PDF, PowerPoint, etc.)

Point clé souvent mal compris : les utilisateurs peuvent utiliser l’agent sans disposer eux-mêmes d'une licence Copilot payante.

❌ Limites majeures :

  • La documentation fournie n'est pas mise à jour automatiquement.
  • Les documents chargés deviennent accessibles à tous les utilisateurs

 

💡 Cas d'usages types : Procédures et guides internes stables dans le temps

Cas n°3 : Copilot pour Microsoft 365 (à grande échelle)

Le créateur et les utilisateurs disposent de licences Copilot pour Microsoft 365.

⭐ Ca y'est, on joue dans la cour des grands !

Le créateur peut créer un agent déclaratif basé sur toute la connaissance de l’entreprise disponible dans le cloud Microsoft.

L’agent peut interroger directement :

  • SharePoint
  • OneDrive
  • Teams
  • et plus largement l’écosystème Microsoft 365

🔐 Tout en respectant strictement les droits d’accès déjà en place !

 

💡 Cas d'usages types : Support informatique de Niveau 1, Onboarding des nouveaux collaborateurs

Et Copilot Studio dans tout ça ?

Copilot Studio permet d’aller plus loin en créant des agents capables de déclencher des actions, d’orchestrer des workflows et d’interagir avec des systèmes métiers. On ne parle plus seulement de répondre, mais d’agir.

En conclusion...

Aujourd’hui, dans la majorité des organisations que j’accompagne, le scénario n°2 est le plus courant.
Les entreprises sont encore en phase de pilote, testent les usages, mesurent la valeur… et avancent progressivement.

👉 Les agents déclaratifs sont souvent la première marche avant :

  • une généralisation des licences,
  • et des scénarios plus avancés intégrant Copilot Studio.
