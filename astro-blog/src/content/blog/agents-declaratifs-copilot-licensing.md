"Est-ce qu'on peut créer des agents Copilot sans licence ?" La réponse courte : oui, on peut créer des agents déclaratifs sans licence Copilot payante, mais les possibilités restent limitées.

## Les différentes licences Copilot

### Copilot Chat (inclus)

Copilot Chat est inclus dans la plupart des licences Microsoft 365. Avec cette option, vous pouvez :

- Accéder à l'interface web de Copilot
- Créer des agents déclaratifs simples
- Tester des cas d'usage basiques

### Copilot pour Microsoft 365

Cette licence payante déverrouille :

- L'intégration native dans Teams, Outlook, Word, Excel
- Des capacités avancées d'agents
- L'accès aux données de votre tenant via Microsoft Graph

## Agents déclaratifs sans licence payante

Un agent déclaratif est essentiellement un prompt enrichi qui donne un contexte et un comportement spécifique à Copilot. Avec Copilot Chat gratuit, vous pouvez créer des agents qui :

- Répondent à des questions sur un sujet précis
- Suivent un ton ou un style défini
- Orientent l'utilisateur vers les bonnes ressources

## Conclusion

Si vous débutez avec les agents Copilot, commencez avec Copilot Chat pour valider vos cas d'usage. Une fois que vous avez identifié de la valeur réelle, le passage à une licence payante sera beaucoup mieux justifié.
