Dans beaucoup d'organisations, l'onboarding des nouveaux collaborateurs repose sur des informations stockées dans un Intranet SharePoint. En pratique, les nouveaux arrivants se perdent souvent dans la documentation.

## Le problème de l'onboarding classique

Vous avez probablement vécu cette situation : une page de bienvenue avec 50 liens, une bibliothèque de documents de 200 fichiers, et un nouveau collègue qui vous pose les mêmes questions toutes les heures.

L'agent Copilot est la solution parfaite pour ce cas d'usage.

## Créer votre agent en 5 minutes

### Étape 1 : Accéder à Copilot Chat

Rendez-vous sur [copilot.microsoft.com](https://copilot.microsoft.com) et connectez-vous avec votre compte Microsoft 365.

### Étape 2 : Créer un nouvel agent

Cliquez sur l'icône de création d'agent dans le menu latéral. Donnez-lui un nom parlant : "Assistant Onboarding RH".

### Étape 3 : Définir les instructions

Écrivez des instructions claires :

```
Tu es l'assistant onboarding de notre entreprise. Ton rôle est d'aider les nouveaux collaborateurs à trouver rapidement les informations dont ils ont besoin.

Tu réponds toujours en français, avec un ton bienveillant et professionnel.

Si tu ne connais pas la réponse, tu orientes l'utilisateur vers le service RH.
```

### Étape 4 : Ajouter vos sources

Connectez les bibliothèques SharePoint contenant votre documentation d'onboarding. L'agent utilisera ces documents comme base de connaissance.

### Étape 5 : Tester et partager

Testez votre agent avec quelques questions typiques des nouveaux arrivants, puis partagez-le avec votre équipe RH.

## Résultat

En 5 minutes, vous avez un assistant qui répond aux questions de vos nouveaux collègues 24h/24, basé sur votre documentation officielle.
